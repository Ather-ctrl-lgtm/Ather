 Find students having marks >= 65
javascript
Copy code
db.Student.find({ Marks: { $gte: 65 } }).pretty()
Explanation:

$gte ? Greater Than or Equal To operator.

 Find student having marks = 55 and Status = "PASS"
javascript
Copy code
db.Student.find({ Marks: 55, Status: "PASS" }).pretty()
Explanation:

Both conditions must be true (logical AND by default).

 Find student whose Stud_id = 12
javascript
Copy code
db.Student.find({ Stud_id: 12 }).pretty()
Explanation:

Returns the record where Stud_id exactly matches 12.

If no such record, it returns an empty result.

 Find students whose marks = 30 or Status = "FAIL"
javascript
Copy code
db.Student.find({
  $or: [{ Marks: 30 }, { Status: "FAIL" }]
}).pretty()
Explanation:

$or ? Returns documents satisfying any one of the conditions.

 Display the collection in descending order by Name
javascript
Copy code
db.Student.find().sort({ Name: -1 }).pretty()
Explanation:

sort({Name: -1}) sorts in descending order.

Use 1 for ascending order.

 Find students who live in Pune or Nashik
javascript
Copy code
db.Student.find({
  Address: { $in: ["Pune", "Nashik"] }
}).pretty()
Explanation:

$in ? Matches any value from the given array.

 Update Status as "PASS" for Stud_id = 5
javascript
Copy code
db.Student.updateOne(
  { Stud_id: 5 },
  { $set: { Status: "PASS" } }
)
Explanation:

updateOne() updates only one matching document.

$set changes the specified field value.

 Delete the record of Amit
javascript
Copy code
db.Student.deleteOne({ Name: "Amit" })
Explanation:

Removes one document where Name is "Amit".

 Skip 3 records and display only 5 records
javascript
Copy code
db.Student.find().skip(3).limit(5).pretty()
